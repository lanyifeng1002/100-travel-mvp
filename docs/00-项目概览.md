# 项目概览

Amazing 100 Journeys 目前已经整理成“源码 + 文档”两部分，结构清晰，适合继续维护和扩展。

## 一、项目源码

核心目录：

- `src`
- `server`
- `tests`
- `public`
- `package.json`
- `package-lock.json`
- `vite.config.js`
- `Dockerfile`
- `.dockerignore`
- `README.md`

数据库初始化脚本：

- `server/db/initDatabase.js`
- `server/scripts/resetDatabase.js`

## 二、文档目录

当前文档包括：

- `docs/01-产品需求.md`
- `docs/02-接口说明.md`
- `docs/03-AI协作记录.md`
- `docs/04-开发说明.md`
- `docs/05-维护指南.md`

## 三、当前能力范围

已完成：

- 目的地首页浏览
- 搜索、标签筛选、随机探索
- 详情页与相关推荐
- 内容管理页
- SQLite 数据持久化
- 自动化测试
- Docker 部署文件

暂未包含：

- 用户登录与权限系统
- 后台编辑、删除和审核能力
- 正式线上环境地址
- 独立云数据库

## 四、建议保留方式

- 使用源码目录管理项目，不保留 `node_modules`
- `dist` 可以由本地重新构建，不需要作为核心源码保存
- 文档可以保留 Markdown，也可以按需要导出 PDF
